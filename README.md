# 🫀 PulseCircle: Home Cardiac Rehab Companion

PulseCircle is a high-fidelity MVP designed to bridge the gap between hospital discharge and long-term cardiac recovery for heart failure patients.

## 🌟 Key Features
- **Nurse Clara AI:** A warm, grandmotherly AI assistant designed for geriatric accessibility.
- **Safety-First Architecture:** AI is restricted to encouragement and explanation; it never provides medical diagnoses.
- **Volunteer Layer:** A moderated community feed where vetted volunteers can send "Digital Get Well" cards.
- **Clinical Escalation:** Automated flagging of high-risk symptoms (chest pain, dizziness) directly to a Care Team dashboard.

## 🛠 Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS (Accessible Color Palette)
- **Animations:** Framer Motion (Calm, slow-paced transitions)
- **Icons:** Lucide React

## ⚠️ Medical Disclaimer
This application is a supportive UX prototype. It does not provide medical advice, diagnosis, or treatment. All clinical decisions must be made by a qualified healthcare professional.
